# Week 2 - Debugging and Troubleshooting Python Applications

## Project Overview

This project was created as part of my Junior Python Developer Internship Week 2 task. The objective of this project was to practice real-world debugging and troubleshooting techniques in Python by identifying, reproducing, documenting, and fixing multiple bugs in a student management application.

The project demonstrates practical debugging skills including runtime error handling, logic error correction, exception handling, file handling improvements, input validation, and code refactoring.

---

## Project Files

* `buggy_student_manager.py`

  * Original buggy version of the application

* `fixed_student_manager.py`

  * Fully corrected and optimized version

* `Bug_Report.md`

  * Detailed report of all identified bugs

* `Debugging_Log.md`

  * Step-by-step debugging investigation and fixes

* `student_data.txt`

  * Stores saved student records

* `screenshots/`

  * Contains screenshots of reproduced bugs and fixes
